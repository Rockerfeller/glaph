Creating node types

For example, let's take the current version of the node that takes a number as input: 

"number": {
    inputs: [],
    outputs: ["number"],
    icon: "fa-calculator",
    settings: {
        number:{
            type: "float",
            value: 0,
        }
    },
    calculate: function(nodes,id){
        var self = nodes[id];
        self.result = [
            parseFloat(nodes[id].settings["number"])
        ];
    }
}

QSA(selector), SQSA(someElement,selector)

They return an array of DOM elements.

You should always use SQSA with g_root.cont in order to make sure your script does not interfere with other instances of graph:

var my_elements = SQSA(g_root.cont,".some-class");
You should limit your use of these to the bare minimum.
(Don't select an element twice. Keep a reference to it in a variable!)
